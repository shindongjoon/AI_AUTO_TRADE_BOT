## Environment

Python = 3.7(32-bit)
(Kium Securities's Open api does not support 64-bit python.)

### How to set python to 32-bit in 64-bit Anaconda

Run below commands in anaconda prompt


```Anaconda
conda create -n ENV
conda activate ENV
conda config --env --set subdir win-32
conda install python=3.7
```
